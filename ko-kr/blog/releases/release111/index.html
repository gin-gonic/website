Gin v1.11.0

새로운 기능

  • feat(gin): quic-go/quic-go를 통한 HTTP/3의 실험적 지원 (#3210)
  • feat(form): 폼 바인딩에서 배열 컬렉션 형식 추가 (#3986), 폼 태그 언마샬을 위한 사용자 지정 문자열 슬라이스 추가 (#3970)
  • feat(binding): BindPlain 추가 (#3904)
  • feat(fs): OnlyFilesFS 내보내기, 테스트 및 문서화 (#3939)
  • feat(binding): unixMilli 및 unixMicro 지원 추가 (#4190)
  • feat(form): 폼 바인딩에서 컬렉션 기본값 지원 (#4048)
  • feat(context): GetXxx에서 Go의 다양한 기본 타입 지원 추가 (#3633)

개선 사항

  • perf(context): getMapFromFormData 성능 최적화 (#4339)
  • refactor(tree): node.insertChild에서 string(/)을 ”/“로 교체 (#4354)
  • refactor(render): writeHeader에서 headers 파라미터 제거 (#4353)
  • refactor(context): “GetType()” 함수 간소화 (#4080)
  • refactor(slice): SliceValidationError Error 메서드 간소화 (#3910)
  • refactor(context): SaveUploadedFile에서 filepath.Dir의 이중 사용 방지 (#4181)
  • refactor(context): 컨텍스트 처리 리팩터링 및 테스트 견고성 향상 (#4066)
  • refactor(binding): strings.Index 대신 strings.Cut 사용 (#3522)
  • refactor(context): SaveUploadedFile에 선택적 권한 파라미터 추가 (#4068)
  • refactor(context): initQueryCache()에서 URL 널 체크 추가 (#3969)
  • refactor(context): Negotiate에서 YAML 판별 로직 개선 (#3966)
  • tree: 자체 min 함수에서 공식 min으로 교체 (#3975)
  • context: filepath.Dir의 불필요한 사용 제거 (#4181)
