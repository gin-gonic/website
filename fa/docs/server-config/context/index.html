ШҜШұЪ© ЩҶШӯЩҲЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ ШөШӯЫҢШӯ Ш§ШІ context ШІЫҢШұШЁЩҶШ§ЫҢЫҢ ШЁШұШ§ЫҢ ШіШ§Ш®ШӘ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ШӘЩҲЩ„ЫҢШҜЫҢ Ъ©ЩҮ timeoutЩҮШ§ШҢ Щ„ШәЩҲ ЩҲ ЩҫШ§Ъ©ШіШ§ШІЫҢ Щ…ЩҶШ§ШЁШ№ ШұШ§ ШЁЩҮ ШҜШұШіШӘЫҢ Щ…ШҜЫҢШұЫҢШӘ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ Ш¶ШұЩҲШұЫҢ Ш§ШіШӘ.

ШҜШіШӘШұШіЫҢ ШЁЩҮ context ШҜШұШ®ЩҲШ§ШіШӘ

context.Context Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШЁШұШ§ЫҢ ШҜШұШ®ЩҲШ§ШіШӘ Ш¬Ш§ШұЫҢ Ш§ШІ Ш·ШұЫҢЩӮ c.Request.Context() ШҜШұ ШҜШіШӘШұШі Ш§ШіШӘ. Ш§ЫҢЩҶ contextЫҢ Ш§ШіШӘ Ъ©ЩҮ ШЁШ§ЫҢШҜ ШЁЩҮ ЩҮШұ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ downstream Ш§ШұШіШ§Щ„ Ъ©ЩҶЫҢШҜ вҖ” ЩҫШұШівҖҢЩҲШ¬ЩҲЩҮШ§ЫҢ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮШҢ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ЫҢ HTTP ЫҢШ§ ШіШ§ЫҢШұ Ш№Щ…Щ„ЫҢШ§ШӘ I/O.

timeout ШҜШұШ®ЩҲШ§ШіШӘ

Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЫҢЪ© Щ…ЫҢШ§ЩҶвҖҢШ§ЩҒШІШ§Шұ timeout ШЁЩҮ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ЫҢ Щ…ЩҶЩҒШұШҜ Ш§Ш№Щ…Ш§Щ„ Ъ©ЩҶЫҢШҜ. ЩҲЩӮШӘЫҢ timeout Щ…ЩҶЩӮШ¶ЫҢ ШҙЩҲШҜШҢ context Щ„ШәЩҲ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ ЩҮШұ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ downstream Ъ©ЩҮ Щ„ШәЩҲ context ШұШ§ ШұШ№Ш§ЫҢШӘ Ъ©ЩҶШҜ ШЁЩ„Ш§ЩҒШ§ШөЩ„ЩҮ ШЁШұЩ…ЫҢвҖҢЪҜШұШҜШҜ.

Ш§ШұШіШ§Щ„ context ШЁЩҮ ЩҫШұШівҖҢЩҲШ¬ЩҲЩҮШ§ЫҢ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ

ШҜШұШ§ЫҢЩҲШұЩҮШ§ЫҢ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ ШҜШұ Go context.Context ШұШ§ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Ш§ЩҲЩ„ЫҢЩҶ ШўШұЪҜЩҲЩ…Ш§ЩҶ Щ…ЫҢвҖҢЩҫШ°ЫҢШұЩҶШҜ. ЩҮЩ…ЫҢШҙЩҮ context ШҜШұШ®ЩҲШ§ШіШӘ ШұШ§ Ш§ШұШіШ§Щ„ Ъ©ЩҶЫҢШҜ ШӘШ§ ЩҫШұШівҖҢЩҲШ¬ЩҲЩҮШ§ ШҜШұ ШөЩҲШұШӘ ЩӮШ·Ш№ Ш§ШӘШөШ§Щ„ Ъ©Щ„Ш§ЫҢЩҶШӘ ЫҢШ§ timeout ШҜШұШ®ЩҲШ§ШіШӘ ШЁЩҮ Ш·ЩҲШұ Ш®ЩҲШҜЪ©Ш§Шұ Щ„ШәЩҲ ШҙЩҲЩҶШҜ.

Ш§ШұШіШ§Щ„ context ШЁЩҮ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ HTTP Ш®ШұЩҲШ¬ЫҢ

ЩҲЩӮШӘЫҢ handler ШҙЩ…Ш§ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Ш®Ш§ШұШ¬ЫҢ ШұШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ context ШҜШұШ®ЩҲШ§ШіШӘ ШұШ§ Ш§ШұШіШ§Щ„ Ъ©ЩҶЫҢШҜ ШӘШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ Ш®ШұЩҲШ¬ЫҢ ЩҮЩ…ШұШ§ЩҮ ШЁШ§ ШҜШұШ®ЩҲШ§ШіШӘ ЩҲШұЩҲШҜЫҢ Щ„ШәЩҲ ШҙЩҲЩҶШҜ.

Щ…ШҜЫҢШұЫҢШӘ ЩӮШ·Ш№ Ш§ШӘШөШ§Щ„ Ъ©Щ„Ш§ЫҢЩҶШӘ

ЩҲЩӮШӘЫҢ Ъ©Щ„Ш§ЫҢЩҶШӘ Ш§ШӘШөШ§Щ„ ШұШ§ Щ…ЫҢвҖҢШЁЩҶШҜШҜ (Щ…Ш«Щ„Ш§ЩӢ ШЁЩҮ ШөЩҒШӯЩҮ ШҜЫҢЪҜШұЫҢ Щ…ЫҢвҖҢШұЩҲШҜ ЫҢШ§ ШҜШұШ®ЩҲШ§ШіШӘ ШұШ§ Щ„ШәЩҲ Щ…ЫҢвҖҢЪ©ЩҶШҜ)ШҢ context ШҜШұШ®ЩҲШ§ШіШӘ Щ„ШәЩҲ Щ…ЫҢвҖҢШҙЩҲШҜ. Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ЫҢЩҶ ШұШ§ ШҜШұ handlerЩҮШ§ЫҢ Ш·ЩҲЩ„Ш§ЩҶЫҢвҖҢЩ…ШҜШӘ ШӘШҙШ®ЫҢШө ШҜЩҮЫҢШҜ ШӘШ§ Ъ©Ш§Шұ ШұШ§ ШІЩҲШҜШӘШұ Щ…ШӘЩҲЩӮЩҒ Ъ©ШұШҜЩҮ ЩҲ Щ…ЩҶШ§ШЁШ№ ШұШ§ ШўШІШ§ШҜ Ъ©ЩҶЫҢШҜ.

ШЁЩҮШӘШұЫҢЩҶ ШұЩҲШҙвҖҢЩҮШ§

  • ЩҮЩ…ЫҢШҙЩҮ context ШҜШұШ®ЩҲШ§ШіШӘ ШұШ§ Щ…ЩҶШӘШҙШұ Ъ©ЩҶЫҢШҜ. c.Request.Context() ШұШ§ ШЁЩҮ ЩҮШұ ШӘШ§ШЁШ№ЫҢ Ъ©ЩҮ context.Context Щ…ЫҢвҖҢЩҫШ°ЫҢШұШҜ Ш§ШұШіШ§Щ„ Ъ©ЩҶЫҢШҜ вҖ” ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮШҢ Ъ©Щ„Ш§ЫҢЩҶШӘвҖҢЩҮШ§ЫҢ HTTPШҢ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ gRPC ЩҲ ЩҮШұ Ш№Щ…Щ„ЫҢШ§ШӘ I/O. Ш§ЫҢЩҶ ШӘШ¶Щ…ЫҢЩҶ Щ…ЫҢвҖҢЪ©ЩҶШҜ Щ„ШәЩҲ ЩҲ timeoutЩҮШ§ ШҜШұ Ъ©Щ„ ШІЩҶШ¬ЫҢШұЩҮ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ЩҶШӘШҙШұ ШҙЩҲЩҶШҜ.

  • *gin.Context ШұШ§ ШҜШұ structЩҮШ§ Ш°Ш®ЫҢШұЩҮ ЩҶЪ©ЩҶЫҢШҜ ЫҢШ§ ШўЩҶ ШұШ§ Ш§ШІ Щ…ШұШІЩҮШ§ЫҢ ЪҜЩҲШұЩҲШӘЫҢЩҶ Ш№ШЁЩҲШұ ЩҶШҜЩҮЫҢШҜ. gin.Context ШЁЩҮ ЪҶШұШ®ЩҮ ШӯЫҢШ§ШӘ ШҜШұШ®ЩҲШ§ШіШӘ/ЩҫШ§ШіШ® HTTP ЩҲШ§ШЁШіШӘЩҮ Ш§ШіШӘ ЩҲ ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҮЩ…ШІЩ…Ш§ЩҶ Ш§ЫҢЩ…ЩҶ ЩҶЫҢШіШӘ. ШҜШұ Ш№ЩҲШ¶ШҢ Щ…ЩӮШ§ШҜЫҢШұ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ Ш®ЩҲШҜ (context ШҜШұШ®ЩҲШ§ШіШӘШҢ ЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§ШҢ ЩҮШҜШұЩҮШ§) ШұШ§ ЩӮШЁЩ„ Ш§ШІ Ш§ЫҢШ¬Ш§ШҜ ЪҜЩҲШұЩҲШӘЫҢЩҶвҖҢЩҮШ§ Ш§ШіШӘШ®ШұШ§Ш¬ Ъ©ЩҶЫҢШҜ.

  • timeout ШұШ§ ШҜШұ ШіШ·Шӯ Щ…ЫҢШ§ЩҶвҖҢШ§ЩҒШІШ§Шұ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ. ЫҢЪ© Щ…ЫҢШ§ЩҶвҖҢШ§ЩҒШІШ§Шұ timeout ЫҢЪ© Щ…Ъ©Ш§ЩҶ ЩҲШ§ШӯШҜ ШЁШұШ§ЫҢ Ш§Ш№Щ…Ш§Щ„ Щ…ЩҮЩ„ШӘвҖҢЩҮШ§ ШҜШұ ШӘЩ…Ш§Щ… Щ…ШіЫҢШұЩҮШ§ ШЁЩҮ ШҙЩ…Ш§ Щ…ЫҢвҖҢШҜЩҮШҜШҢ ШЁЩҮ Ш¬Ш§ЫҢ ШӘЪ©ШұШ§Шұ Щ…ЩҶШ·ЩӮ timeout ШҜШұ ЩҮШұ handler.

  • Ш§ШІ context.WithValue ШЁШ§ Ш§ШӯШӘЫҢШ§Ш· Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. c.Set() ЩҲ c.Get() ШұШ§ ШҜШұ handlerЩҮШ§ЫҢ Gin ШӘШұШ¬ЫҢШӯ ШҜЩҮЫҢШҜ. context.WithValue ШұШ§ ШЁШұШ§ЫҢ Щ…ЩӮШ§ШҜЫҢШұЫҢ Ъ©ЩҮ ШЁШ§ЫҢШҜ Ш§ШІ Щ…ШұШІЩҮШ§ЫҢ ЩҫЪ©ЫҢШ¬ Ш§ШІ Ш·ШұЫҢЩӮ ШұШ§ШЁШ·вҖҢЩҮШ§ЫҢ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ Ш№ШЁЩҲШұ Ъ©ЩҶЩҶШҜ ЩҶЪҜЩҮ ШҜШ§ШұЫҢШҜ.

Щ…ШҙЪ©Щ„Ш§ШӘ ШұШ§ЫҢШ¬

Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ gin.Context ШҜШұ ЪҜЩҲШұЩҲШӘЫҢЩҶвҖҢЩҮШ§

gin.Context ШЁШұШ§ЫҢ Ш№Щ…Щ„Ъ©ШұШҜ ШҜШұ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ ШЁШ§ШІШ§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЪҜШұ ЩҶЫҢШ§ШІ ШҜШ§ШұЫҢШҜ Ш§ШІ ЫҢЪ© ЪҜЩҲШұЩҲШӘЫҢЩҶ ШЁЩҮ ШўЩҶ ШҜШіШӘШұШіЫҢ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜШҢ ШЁШ§ЫҢШҜ c.Copy() ШұШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Ъ©ЩҶЫҢШҜ ШӘШ§ ЫҢЪ© Ъ©ЩҫЫҢ ЩҒЩӮШ· Ш®ЩҲШ§ЩҶШҜЩҶЫҢ Ш§ЫҢШ¬Ш§ШҜ Ъ©ЩҶЫҢШҜ. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ gin.Context Ш§ШөЩ„ЫҢ ШҜШұ ЫҢЪ© ЪҜЩҲШұЩҲШӘЫҢЩҶ Щ…ЩҶШ¬Шұ ШЁЩҮ ШҙШұШ§ЫҢШ· ШұЩӮШ§ШЁШӘЫҢ ЩҲ ШұЩҒШӘШ§Шұ ШәЫҢШұЩӮШ§ШЁЩ„ ЩҫЫҢШҙвҖҢШЁЫҢЩҶЫҢ Щ…ЫҢвҖҢШҙЩҲШҜ.

ЩҶШ§ШҜЫҢШҜЩҮ ЪҜШұЩҒШӘЩҶ Щ„ШәЩҲ context

Ш§ЪҜШұ handler ШҙЩ…Ш§ ctx.Done() ШұШ§ ШЁШұШұШіЫҢ ЩҶЪ©ЩҶШҜШҢ ШӯШӘЫҢ ЩҫШі Ш§ШІ ЩӮШ·Ш№ Ш§ШӘШөШ§Щ„ Ъ©Щ„Ш§ЫҢЩҶШӘ ШЁЩҮ Ш§Ш¬ШұШ§ Ш§ШҜШ§Щ…ЩҮ Ш®ЩҲШ§ЩҮШҜ ШҜШ§ШҜ ЩҲ CPU ЩҲ ШӯШ§ЩҒШёЩҮ ЩҮШҜШұ Щ…ЫҢвҖҢШҜЩҮШҜ. ЩҮЩ…ЫҢШҙЩҮ Ш§ШІ APIЩҮШ§ЫҢ ШўЪҜШ§ЩҮ Ш§ШІ context (QueryRowContextШҢ NewRequestWithContextШҢ select ШұЩҲЫҢ ctx.Done()) Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ШӘШ§ Ъ©Ш§Шұ ШЁЩҮ Щ…ШӯШ¶ Щ„ШәЩҲ context Щ…ШӘЩҲЩӮЩҒ ШҙЩҲШҜ.

ЩҶЩҲШҙШӘЩҶ ЩҫШ§ШіШ® ЩҫШі Ш§ШІ Щ„ШәЩҲ context

ЩҫШі Ш§ШІ Щ„ШәЩҲ contextШҢ Ш§ШІ ЩҶЩҲШҙШӘЩҶ ШҜШұ c.Writer Ш®ЩҲШҜШҜШ§ШұЫҢ Ъ©ЩҶЫҢШҜ. Ш§ШӘШөШ§Щ„ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩӮШЁЩ„Ш§ЩӢ ШЁШіШӘЩҮ ШҙШҜЩҮ ШЁШ§ШҙШҜ ЩҲ ЩҶЩҲШҙШӘЩҶвҖҢЩҮШ§ ШЁЫҢвҖҢШөШҜШ§ ШҙЪ©ШіШӘ Щ…ЫҢвҖҢШ®ЩҲШұЩҶШҜ ЫҢШ§ panic Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. ЩӮШЁЩ„ Ш§ШІ ЩҶЩҲШҙШӘЩҶШҢ ctx.Err() ШұШ§ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢШҜ Ш§ЪҜШұ handler ШҙЩ…Ш§ Ъ©Ш§Шұ Ш·ЩҲЩ„Ш§ЩҶЫҢвҖҢЩ…ШҜШӘ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҜЩҮШҜ.

ЩҲЫҢШұШ§ЫҢШҙ ШөЩҒШӯЩҮ

ШўШ®ШұЫҢЩҶ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ: ЫұЫҙ ЩҒШұЩҲШұШҜЫҢЩҶ ЫұЫҙЫ°Ыө

ЩӮШЁЩ„ЫҢ
ЫҢЪ©ЩҫШ§ШұЪҶЩҮвҖҢШіШ§ШІЫҢ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ ШЁШ№ШҜЫҢ
ШЁШұШұШіЫҢ ШіЩ„Ш§Щ…ШӘ
