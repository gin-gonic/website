優雅地重新啟動或停止

您想要優雅地重新啟動或停止您的網頁伺服器嗎？ 有幾種方法可以做到。

我們可以使用 fvbock/endless 來取代預設的 ListenAndServe。詳情請參閱問題 #296。

endless 的替代方案：

  • manners：一個能優雅關閉的 Go HTTP 伺服器。
  • graceful：Graceful 是一個 Go 套件，可讓 http.Handler 伺服器優雅關閉。
  • grace：Go 伺服器的優雅重啟與零停機部署。

如果您使用 Go 1.8 或更新版本，您可能不需要使用此函式庫！請考慮使用 http.Server 內建的 Shutdown() 方法來進行優雅關閉。請參閱 gin 的完整優雅關閉範例。
