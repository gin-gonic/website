## Gin 是什麼？

Gin 是一個使用 Go (Golang) 寫的 HTTP Web 框架。它提供類似 Martini 的 API，但擁有更好的效能——速度快上了 40 倍。如果你需要震撼性的效能，給自己來點 Gin 吧。

我們提供了一些 API 使用 [範例](https://github.com/gin-gonic/examples) 並列出一些知名的 [Gin 使用者](./users).
