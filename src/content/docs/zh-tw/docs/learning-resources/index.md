## 官方資源

這些由 Gin 團隊維護，是最具權威性的資訊來源。

- **[Gin GitHub 儲存庫](https://github.com/gin-gonic/gin)** -- 原始碼、問題追蹤器和最新的發布說明。
- **[Gin 範例儲存庫](https://github.com/gin-gonic/examples)** -- 涵蓋常見使用情境的獨立範例專案集合，如檔案上傳、優雅關閉、JWT 身份驗證等。
- **[Gin API 參考（pkg.go.dev）](https://pkg.go.dev/github.com/gin-gonic/gin)** -- 每個匯出的類型和函式的自動產生 API 文件。

## Go 語言資源

對 Go 語言本身的紮實理解將使使用 Gin 變得更加容易。

- **[A Tour of Go](https://go.dev/tour/)** -- 官方的 Go 互動式入門教程。
- **[Effective Go](https://go.dev/doc/effective_go)** -- 撰寫 Go 程式碼的最佳實務和慣用模式。
- **[Go by Example](https://gobyexample.com/)** -- 涵蓋 Go 基礎的帶註解範例程式。
- **[Go 標準函式庫文件](https://pkg.go.dev/std)** -- 標準函式庫中每個套件的參考，包括 Gin 所建構的 `net/http`。

## 教學與文章

由社群撰寫的指南，引導你使用 Gin 建構真實應用程式。

- **[使用 Go 和 Gin 建構 RESTful API（Go.dev）](https://go.dev/doc/tutorial/web-service-gin)** -- 官方 Go 教學，使用 Gin 建構簡單的 Web 服務。
- **[使用 Go、Gin 和 GORM 建構 REST API（Digital Ocean）](https://www.digitalocean.com/community/tutorials/how-to-build-and-deploy-a-go-application)** -- 涵蓋專案佈局、資料庫整合和部署。
