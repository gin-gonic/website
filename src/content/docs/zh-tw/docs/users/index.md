Gin 被各種規模的企業和開源專案在正式環境中使用。以下是使用 Gin 建構的知名專案精選。如果你的專案使用 Gin，我們很樂意將其加入此列表。

## 優秀專案列表

- [gorush](https://github.com/appleboy/gorush)：一個用 Go 撰寫的推播通知伺服器。
- [fnproject](https://github.com/fnproject/fn)：容器原生、雲端無關的無伺服器平台。
- [photoprism](https://github.com/photoprism/photoprism)：去中心化網路的 AI 驅動照片應用程式。
- [krakend](https://github.com/devopsfaith/krakend)：帶有中介軟體的超高效能 API 閘道。
- [picfit](https://github.com/thoas/picfit)：一個用 Go 撰寫的圖片調整大小伺服器。
- [gotify](https://github.com/gotify/server)：一個透過 WebSocket 即時傳送和接收訊息的簡單伺服器。
- [cds](https://github.com/ovh/cds)：企業級持續交付和 DevOps 自動化開源平台。
- [drone](https://github.com/harness/drone)：一個基於 Go 建構的容器原生持續交付平台。
- [go-admin](https://github.com/GoAdminGroup/go-admin)：一個 Go 的資料視覺化和管理面板框架。

## 提交你的專案

在正式環境或開源專案中使用 Gin？在 [gin-gonic/website](https://github.com/gin-gonic/website) 儲存庫上開一個 pull request，將你的專案加入此頁面。
