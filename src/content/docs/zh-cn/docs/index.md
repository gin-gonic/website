## Gin 是什么？

Gin 是一个用 Go (Golang) 编写的 HTTP Web 框架。 它具有类似 Martini 的 API，但性能比 Martini 快 40 倍。如果你需要极好的性能，使用 Gin 吧。

## 如何使用 Gin？

我们提供了一些 API 的使用[示例](https://gin-gonic.com/zh-cn/docs/examples/) 并罗列了一些众所周知的 [Gin 用户](./users).

* 告诉我们你使用 Gin 的成功案例
