## 官方资源

以下资源由 Gin 团队维护，是最权威的信息来源。

- **[Gin GitHub 仓库](https://github.com/gin-gonic/gin)** -- 源代码、问题追踪和最新发布说明。
- **[Gin 示例仓库](https://github.com/gin-gonic/examples)** -- 一组独立的示例项目，涵盖文件上传、优雅关闭、JWT 认证等常见用例。
- **[Gin API 参考（pkg.go.dev）](https://pkg.go.dev/github.com/gin-gonic/gin)** -- 每个导出类型和函数的自动生成 API 文档。

## Go 语言资源

扎实的 Go 语言基础会让使用 Gin 变得更加容易。

- **[Go 之旅](https://go.dev/tour/)** -- Go 的官方交互式入门教程。
- **[Effective Go](https://go.dev/doc/effective_go)** -- 编写 Go 代码的最佳实践和惯用模式。
- **[Go by Example](https://gobyexample.com/)** -- 带注释的示例程序，涵盖 Go 基础知识。
- **[Go 标准库文档](https://pkg.go.dev/std)** -- 标准库中每个包的参考文档，包括 Gin 构建于其上的 `net/http`。

## 教程和文章

社区编写的指南，带你逐步构建使用 Gin 的实际应用。

- **[使用 Go 和 Gin 构建 RESTful API（Go.dev）](https://go.dev/doc/tutorial/web-service-gin)** -- 使用 Gin 构建简单 Web 服务的 Go 官方教程。
- **[使用 Go、Gin 和 GORM 构建 REST API（Digital Ocean）](https://www.digitalocean.com/community/tutorials/how-to-build-and-deploy-a-go-application)** -- 涵盖项目结构、数据库集成和部署。
