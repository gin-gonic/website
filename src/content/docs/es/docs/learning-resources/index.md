## Recursos oficiales

Estos son mantenidos por el equipo de Gin y son las fuentes de información más autorizadas.

- **[Repositorio GitHub de Gin](https://github.com/gin-gonic/gin)** -- Código fuente, rastreador de issues y las últimas notas de lanzamiento.
- **[Repositorio de ejemplos de Gin](https://github.com/gin-gonic/examples)** -- Una colección de proyectos de ejemplo autocontenidos que cubren casos de uso comunes como subida de archivos, parada elegante, autenticación JWT y más.
- **[Referencia API de Gin (pkg.go.dev)](https://pkg.go.dev/github.com/gin-gonic/gin)** -- Documentación API generada automáticamente para cada tipo y función exportados.

## Recursos del lenguaje Go

Un conocimiento sólido de Go en sí facilitará mucho el trabajo con Gin.

- **[Un tour por Go](https://go.dev/tour/)** -- La introducción interactiva oficial a Go.
- **[Go efectivo](https://go.dev/doc/effective_go)** -- Mejores prácticas y patrones idiomáticos para escribir código Go.
- **[Go mediante ejemplos](https://gobyexample.com/)** -- Programas de ejemplo anotados que cubren los fundamentos de Go.
- **[Documentación de la biblioteca estándar de Go](https://pkg.go.dev/std)** -- Referencia para cada paquete de la biblioteca estándar, incluyendo `net/http`, sobre el cual Gin se construye.

## Tutoriales y artículos

Guías escritas por la comunidad que te llevan paso a paso a través de la construcción de aplicaciones reales con Gin.

- **[Construyendo una API RESTful con Go y Gin (Go.dev)](https://go.dev/doc/tutorial/web-service-gin)** -- Un tutorial oficial de Go que usa Gin para construir un servicio web simple.
- **[Construir una API REST con Go, Gin y GORM (Digital Ocean)](https://www.digitalocean.com/community/tutorials/how-to-build-and-deploy-a-go-application)** -- Cubre la estructura del proyecto, integración con base de datos y despliegue.
