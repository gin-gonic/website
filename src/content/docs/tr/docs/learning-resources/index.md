## Resmi Kaynaklar

Bunlar Gin ekibi tarafından sürdürülür ve en yetkili bilgi kaynaklarıdır.

- **[Gin GitHub Deposu](https://github.com/gin-gonic/gin)** -- Kaynak kodu, sorun takipçisi ve en son sürüm notları.
- **[Gin Örnekler Deposu](https://github.com/gin-gonic/examples)** -- Dosya yükleme, zarif kapatma, JWT kimlik doğrulama ve daha fazlası gibi yaygın kullanım senaryolarını kapsayan bağımsız örnek projeler koleksiyonu.
- **[Gin API Referansı (pkg.go.dev)](https://pkg.go.dev/github.com/gin-gonic/gin)** -- Dışa aktarılan her tür ve fonksiyon için otomatik oluşturulan API belgesi.

## Go Dili Kaynakları

Go'nun kendisini iyi anlamak, Gin ile çalışmayı çok daha kolay hale getirecektir.

- **[Go Turu](https://go.dev/tour/)** -- Go'ya resmi etkileşimli giriş.
- **[Etkili Go](https://go.dev/doc/effective_go)** -- Go kodu yazmak için en iyi uygulamalar ve deyimsel kalıplar.
- **[Örneklerle Go](https://gobyexample.com/)** -- Go temellerini kapsayan açıklamalı örnek programlar.
- **[Go Standart Kütüphane Belgesi](https://pkg.go.dev/std)** -- Gin'in üzerine inşa edildiği `net/http` dahil, standart kütüphanedeki her paket için referans.

## Eğitimler ve Makaleler

Gin ile gerçek uygulamalar oluşturma sürecinde yol gösteren topluluk tarafından yazılmış kılavuzlar.

- **[Go ve Gin ile RESTful API Oluşturma (Go.dev)](https://go.dev/doc/tutorial/web-service-gin)** -- Basit bir web servisi oluşturmak için Gin kullanan resmi Go eğitimi.
- **[Go, Gin ve GORM ile REST API Oluşturma (Digital Ocean)](https://www.digitalocean.com/community/tutorials/how-to-build-and-deploy-a-go-application)** -- Proje düzeni, veritabanı entegrasyonu ve dağıtımı kapsar.
