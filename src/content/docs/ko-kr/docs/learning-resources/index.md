## 공식 자료

Gin 팀이 관리하며 가장 권위 있는 정보 소스입니다.

- **[Gin GitHub 저장소](https://github.com/gin-gonic/gin)** -- 소스 코드, 이슈 트래커, 최신 릴리스 노트.
- **[Gin 예제 저장소](https://github.com/gin-gonic/examples)** -- 파일 업로드, 우아한 종료, JWT 인증 등 일반적인 사용 사례를 다루는 독립적인 예제 프로젝트 모음.
- **[Gin API 레퍼런스 (pkg.go.dev)](https://pkg.go.dev/github.com/gin-gonic/gin)** -- 모든 내보낸 타입과 함수에 대한 자동 생성 API 문서.

## Go 언어 자료

Go 자체에 대한 탄탄한 이해가 있으면 Gin 작업이 훨씬 쉬워집니다.

- **[A Tour of Go](https://go.dev/tour/)** -- Go에 대한 공식 대화형 소개.
- **[Effective Go](https://go.dev/doc/effective_go)** -- Go 코드 작성의 모범 사례와 관용적 패턴.
- **[Go by Example](https://gobyexample.com/)** -- Go 기본 사항을 다루는 주석이 달린 예제 프로그램.
- **[Go 표준 라이브러리 문서](https://pkg.go.dev/std)** -- Gin이 기반으로 하는 `net/http`를 포함한 표준 라이브러리의 모든 패키지에 대한 레퍼런스.

## 튜토리얼 및 글

Gin으로 실제 애플리케이션을 구축하는 과정을 안내하는 커뮤니티 작성 가이드입니다.

- **[Go와 Gin으로 RESTful API 구축하기 (Go.dev)](https://go.dev/doc/tutorial/web-service-gin)** -- Gin을 사용하여 간단한 웹 서비스를 구축하는 공식 Go 튜토리얼.
- **[Go, Gin, GORM으로 REST API 구축하기 (Digital Ocean)](https://www.digitalocean.com/community/tutorials/how-to-build-and-deploy-a-go-application)** -- 프로젝트 구조, 데이터베이스 통합, 배포를 다룹니다.
